Pizza Order Delivery App

Project Overview

The Pizza Order Delivery App is a full-stack web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). This app allows users to browse a menu, place pizza orders, and track the delivery status in real-time. It showcases essential features of modern web applications, including user authentication, dynamic routing, and backend API integration.

Features
User Registration and Login 
Browse a dynamic menu of pizzas
Add items to a shopping cart
Place orders with customizable options (e.g., pizza size, toppings)
Real-time order tracking
Admin dashboard for managing orders and menu items

Tech Stack

Frontend
React.js: For building the user interface
Redux: For state management
Axios: For making API requests
CSS/Bootstrap: For responsive and styled UI

Backend
Node.js: Server-side runtime
Express.js: For building RESTful APIs
Mongoose: For modeling and interacting with MongoDB

Database
MongoDB: NoSQL database for storing user, menu, and order data

Tools
Postman: For API testing

Git/GitHub: Version control
